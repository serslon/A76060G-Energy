# Руководство по настройке A7670G Energy Project

## Описание проекта
Этот проект настроен для работы с модемом A7670G от LilyGo на базе ESP32.

## Настройки проекта

### Текущая конфигурация
- **Плата**: T-A7670X (ESP32 + A7670G)
- **Окружение PlatformIO**: `T-A7670X`
- **Активный код**: `src/main.cpp` с модулями мониторинга
- **Порт**: `/dev/cu.usbserial-589A0122971`
- **Скорость**: 115200 бод
- **Интервал мониторинга**: 10 секунд

## Функциональность системы

### Мониторинг батареи
- Напряжение Li-ion батареи (3.0V - 4.2V)
- Процент заряда с автоматическим расчетом
- Статус: Excellent, Good, Fair, Low, Critical
- Предупреждения при заряде ниже 20%

### Мониторинг GSM сети  
- Качество сигнала (0-31) и конвертация в dBm
- Статус регистрации в сети
- Название оператора связи  
- Тип сети (GSM/LTE)
- Предупреждения при потере связи

### Мониторинг GPS позиционирования
- Координаты (широта, долгота) с высокой точностью
- Высота над уровнем моря
- Скорость и направление движения
- Количество видимых спутников
- Точность позиционирования (HDOP)
- Автоматическое включение GPS модуля

### Характеристики модема A7670G
Согласно документации, A7670G поддерживает:
- ✅ GPS (в зависимости от версии)
- ✅ Phone Call 
- ✅ SMS
- ✅ LTE-FDD и LTE-TDD диапазоны
- ✅ GSM: 850/900/1800/1900

> **Важно**: A7670G модель может не иметь встроенного GPS функционала. Если вы заказывали A7670G с GPS, возможно потребуется внешний GPS модуль.

## Команды для работы

### Компиляция
```bash
cd /Users/serslon/Projects/SerSlon/A76060G-Energy
.venv/bin/pio run --environment T-A7670X
```

### Загрузка прошивки
```bash
.venv/bin/pio run --target upload --environment T-A7670X --upload-port /dev/cu.usbserial-589A0122971
```

### Мониторинг последовательного порта
```bash
.venv/bin/pio device monitor --port /dev/cu.usbserial-589A0122971 --baud 115200
```

## Основные AT команды для тестирования

После подключения к монитору порта вы можете отправлять следующие команды:

```
AT                    ; Проверка связи с модемом
AT+CSQ                ; Проверка качества сигнала
AT+CREG?              ; Статус регистрации в сети
AT+COPS?              ; Текущий оператор
AT+CBC                ; Напряжение батареи
AT+SIMCOMATI          ; Информация о прошивке и версии модема
AT+CGNSPWR=1          ; Включить GPS модуль
AT+CGNSINF            ; Получить информацию о GPS
AT+CGNSPWR=0          ; Выключить GPS модуль
```

## Распиновка T-A7670X

| Функция                | GPIO | Описание                    |
|------------------------|------|-----------------------------|
| Peripheral power control | 12   | Управление питанием периферии |
| Modem TX               | 26   | Передача данных модема      |
| Modem RX               | 27   | Прием данных модема         |
| Modem PWR              | 4    | Включение модема            |
| Modem RESET            | 5    | Сброс модема                |
| Modem RING             | 33   | Сигнал входящего вызова     |
| Modem DTR              | 25   | Data Terminal Ready         |
| Battery ADC Pin        | 35   | Мониторинг напряжения батареи |
| Solar ADC Pin          | 36   | Мониторинг солнечной панели |

## Известные особенности

1. **ESP32 V1.x версия T-A7670X**:
   - При питании от батареи необходимо установить BOARD_POWERON_PIN (IO12) в HIGH после старта ESP32
   - При питании от USB-C модем не может перейти в режим сна

2. **SD карта**: При вставленной SD карте невозможно загрузить новую прошивку в Arduino IDE. Извлеките SD карту перед загрузкой.

3. **SIM карта**: Вставьте SIM карту перед включением питания для корректного обнаружения.

## Структура проекта

Проект теперь имеет более организованную структуру:

### Основные файлы проекта (в src/)
- `src/main.cpp` - Основной файл проекта (ранее ATdebug.ino)
- `src/utilities.h` - Конфигурация для платы A7670G

### Файлы, которые остаются без изменений
- `lib/` - Библиотеки проекта (остаются на месте)
- `examples/` - Примеры кода для различных функций (для справки)
- `datasheet/` - Документация и спецификации
- `schematic/` - Схемы устройства
- `docs/` - Документация проекта
- `firmware/` - Предкомпилированные прошивки
- `boards/` - Определения плат
- `dimensions/` - Размеры и чертежи
- `images/` - Изображения для документации
- `platformio.ini` - Конфигурация проекта

### Переключение между примерами

Теперь для использования функций из примеров, вместо изменения `src_dir` в `platformio.ini`, скопируйте нужный код из папки `examples/` в `src/main.cpp`:

```bash
# Например, для использования GPS функциональности:
cp examples/GPS_BuiltIn/GPS_BuiltIn.ino src/main.cpp

# Для использования MQTT:
cp examples/MqttClient/MqttClient.ino src/main.cpp

# Для отправки SMS:
cp examples/SendSMS/SendSMS.ino src/main.cpp
```

После копирования не забудьте также скопировать соответствующий `utilities.h` если он отличается:
```bash
cp examples/GPS_BuiltIn/utilities.h src/
```
