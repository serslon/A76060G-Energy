# Исходный код проекта A7670G Energy

Эта папка содержит основной исходный код проекта.

## Файлы

- `main.cpp` - Основной файл проекта с логикой инициализации и работы с модемом A7670G, мониторингом батареи, GSM сети и GPS
- `utilities.h` - Конфигурационный файл с настройками пинов и определениями для платы T-A7670X
- `battery_monitor.h/.cpp` - Модуль мониторинга батареи
- `gsm_monitor.h/.cpp` - Модуль мониторинга GSM сети
- `gps_monitor.h/.cpp` - Модуль мониторинга GPS позиционирования

## Функциональность

### Основные возможности:
- **AT команды** - Отладка и управление модемом A7670G
- **Мониторинг батареи** - Отображение напряжения, процента заряда и статуса каждые 10 секунд
- **Мониторинг GSM сети** - Отображение оператора, силы сигнала, статуса регистрации каждые 10 секунд
- **Мониторинг GPS** - Отображение координат, высоты, скорости и количества спутников каждые 10 секунд
- **Предупреждения** - Автоматические уведомления при критических состояниях

### Мониторинг батареи:
- Измерение напряжения через ADC (GPIO35)
- Расчет процента заряда для Li-ion батареи (3.0V - 4.2V)
- Статус батареи: Excellent, Good, Fair, Low, Critical
- Предупреждение при заряде ниже 20%

### Мониторинг GSM сети:
- Качество сигнала (0-31, конвертация в dBm)
- Статус регистрации в сети
- Название оператора связи
- Тип сети (GSM/LTE)
- Предупреждение при потере связи с сетью

### Мониторинг GPS:
- Координаты (широта, долгота) с точностью до 6 знаков
- Высота над уровнем моря
- Скорость перемещения
- Направление движения
- Количество видимых спутников
- HDOP (точность позиционирования)
- Дата и время UTC
- Статус GPS фиксации

## Использование

Для изменения функциональности проекта отредактируйте `main.cpp` или скопируйте код из нужного примера:

```bash
# Пример: копирование GPS функциональности
cp ../examples/GPS_BuiltIn/GPS_BuiltIn.ino main.cpp
cp ../examples/GPS_BuiltIn/utilities.h .
```

## Компиляция

После изменений компилируйте проект из корневой папки:

```bash
cd ..
.venv/bin/pio run --environment T-A7670X
```

## Загрузка

Для загрузки прошивки на устройство:

```bash
cd ..
.venv/bin/pio run --target upload --environment T-A7670X --upload-port /dev/cu.usbserial-589A0122971
```
